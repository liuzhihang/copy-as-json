复制JavaBean为Json

特征
----

- 复制JavaBean为Json
    

安装
----
- **在线安装:**
  - `File` -> `Setting` -> `Plugins` -> 搜索 `Copy as Json`

- **手动安装:**
  - [下载插件](https://github.com/liuzhihangs/copy-as-json/releases) -> `File` -> `Setting` -> `Plugins` -> `Install Plugin from Disk...`

使用
----
- 右键菜单选择 `Copy as Json`
- 快捷键 `ctrl shift J`
